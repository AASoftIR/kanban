## ğŸ“„ PDF Export

### Method 1: Export Button
Click the **"Export PDF"** button in the top-right corner. The presentation will capture each slide and generate a PDF file.

### Method 2: Browser Print (Alternative)
1. Press **Ctrl+P** (Windows) or **Cmd+P** (Mac)
2. Select "Save as PDF" as destination
3. Set layout to **Landscape**
4. Print

### Method 3: Reveal.js Print Stylesheet
Add `?print-pdf` to the URL and use browser print:
```
file:///path/to/presentation/index.html?print-pdf
```
