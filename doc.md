### ğŸ”¢ How `range_days` Parameter Works

| User Selects | What Happens          | API Calls Made                                     |
| ------------ | --------------------- | -------------------------------------------------- |
| **1 day**    | Only today's forecast | 1 current + 1 forecast + 7 history = 9 calls       |
| **3 days**   | Today + 2 more days   | 1 current + 3 forecast days + 7 history = 9 calls  |
| **7 days**   | Full week forecast    | 1 current + 7 forecast days + 7 history = 9 calls  |
| **10 days**  | Maximum forecast      | 1 current + 10 forecast days + 7 history = 9 calls |

**Important:** The historical fetch is **always 7 days** regardless of forecast days!
